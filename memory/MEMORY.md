# Memory Index

- [Regla RSI — Ambos lados de 50](feedback_rsi_ambos_50.md) — BUY válido solo si purple > yellow Y ambos > 50; SELL válido solo si purple < yellow Y ambos < 50

- [Calidad de entrada RSI — Patrón 1 vs Patrón 2](feedback_rsi_entry_quality.md) — Patrón 1 (yellow cruza 50 por primera vez) = alta prob; Patrón 2 (yellow ya está del lado correcto, necesita reset) = menor prob
- [Template OBLIGATORIO — 🚦 Decisión actual](feedback_decision_template.md) — Formato compacto con 2 filtros (CDX + RSI 3m), 2 condiciones de entrada y TPs al final con 🐤
- [Zonas críticas RSI Purple — Límites de entrada](feedback_rsi_purple_zones.md) — Purple 50→35 (SELL) / 50→65 (BUY) = zona activa; <35/>65 = riesgo; <30/>70 = agotamiento NO TRADE. Condición de entrada = comportamiento RSI, no nivel de precio.
