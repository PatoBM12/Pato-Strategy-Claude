La condición RSI 3m para entrada válida requiere que AMBOS purple Y yellow estén del mismo lado de 50:

**BUY →** purple > yellow **Y ambos > 50** → target = liquidez arriba (iFVG / EQH)
**SELL →** purple < yellow **Y ambos < 50** → target = liquidez abajo (iFVG / EQL)

**Why:** Cuando ambos están del mismo lado de 50, el momentum es claro y consistente. Si solo purple cruza yellow pero el otro sigue del lado contrario de 50, la señal es débil y puede ser un fake.

**How to apply:** Siempre verificar que AMBOS valores RSI 3m estén encima de 50 (BUY) o debajo de 50 (SELL) antes de confirmar entrada. No es suficiente que purple > yellow — ambos tienen que estar del mismo lado de 50.
